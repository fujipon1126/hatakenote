# hatakenote

家庭菜園・貸し農園向けの畑管理Androidアプリ

## 概要

区画ごとの作付け記録や作業履歴を管理し、連作障害の回避やリマインダー通知で効率的な畑仕事をサポートします。

## 主な機能

### 区画管理
- グリッド形式で畑全体のレイアウトを視覚的に表示
- 区画ごとの作物を色分けで識別（科別に色分け）

### 作付け・作業記録
- 作物、植えた日、収穫日、写真を記録
- 耕起・元肥・追肥などの作業履歴を管理
- 1区画に複数作物の混植にも対応

### 連作障害アラート
- 過去の作付け履歴をもとに相性の悪い作物を警告
- 科ごとの連作NG期間を考慮した提案

### リマインダー
- 追肥時期を自動計算して事前に通知

### カレンダー
- 月表示カレンダーで作業日を一覧
- 天気情報を表示（Open-Meteo API連携）

### AIアシスタント（Gemini連携）
- 育て方の質問・アドバイス
- 写真から病害虫を診断
- 連作障害を考慮した次の作物のおすすめ提案

## 動作環境

- Android 13 (API 33) 以上

## 開発環境

- Kotlin
- Gradle (Kotlin DSL)
- Java 11

## ビルド方法

```bash
./gradlew build
```
